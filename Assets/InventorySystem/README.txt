Inventory System Source Project

This project contains an example of Inventory System with Item pickup and UI Drag & Drop.

- Open 'TestScene' from Scenes folder to launch an example scene
- Press Play: use W,A,S,D to move and Mouse View to look around
- Prefabs Folder contains the pickable items, drop them anywhere in the Scene to pick them in the game
- Items are picked by pointing at them, then pressing F
- Items are dropped by opening the Inventory menu (press Tab) and dragging them out of menu area
